# Аштанга-хридая-самхита — Интерактивный читатель

Веб-приложение для чтения классического аюрведического канона «Аштанга-хридая-самхита» Шримад Вагбхаты в переводе Ю.В. Сорокиной с комментариями И.И. Ветрова.

![Preview](https://img.shields.io/badge/язык-JavaScript-yellow) ![License](https://img.shields.io/badge/license-MIT-green) ![Size](https://img.shields.io/badge/зависимости-нет-blue)

## О проекте

«Аштанга-хридая-самхита» (VII в.) — один из трёх великих канонов аюрведической медицины. Текст охватывает фундаментальные принципы аюрведы: физиологию, диагностику, терапию, фармакологию и духовные практики.

Приложение структурирует 804 текстовых блока из 23 глав по 8 разделам (стханам) и предоставляет интерактивный инструмент для изучения текста с глоссарием санскритских терминов.

## Возможности

- **Навигация по стханам и главам** — 8 разделов, 23 главы, сворачиваемые группы
- **Разметка текста** — стихи (шлоки), комментарии и основной текст визуально разделены
- **Глоссарий санскритских терминов** — 80 терминов с определениями и транслитерацией; всплывают при наведении на подсвеченное слово
- **Полнотекстовый поиск** — поиск по всем 23 главам в реальном времени с подсветкой совпадений
- **Тёмная и светлая тема** — переключается кнопкой, сохраняется в localStorage
- **URL-навигация** — прямые ссылки на конкретные главы через хэш (#ch0 … #ch22)

## Стек

Чистый HTML/CSS/JavaScript без зависимостей и сборщиков. Данные хранятся в `data.js` как ES-модуль.

```
index.html   — разметка и структура
style.css    — дизайн-система (CSS custom properties, светлая/тёмная тема)
app.js       — логика навигации, рендеринга, поиска и тултипов
glossary.js  — 80 санскритских терминов + Unicode-корректный regex
data.js      — 23 главы, 804 блока текста, структурированные из DOCX
```

## Запуск

Приложение использует ES-модули, поэтому требует HTTP-сервер (не `file://`).

```bash
# Python (встроен в macOS/Linux)
python3 -m http.server 8000

# Затем открыть в браузере
open http://localhost:8000
```

## Скриншоты

### Светлая тема — чтение главы со стихами и комментариями
Стихи выделены рамкой с янтарной полосой, комментарии — серым отступом.

### Тёмная тема
Переключается кнопкой в шапке сайдбара.

### Глоссарий
Санскритские термины подчёркнуты пунктиром; при наведении — тултип с определением и оригинальной транслитерацией.
