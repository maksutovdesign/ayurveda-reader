# Аштанга-хридая-самхита — Интерактивный читатель ауюрведы

Веб-приложение для изучения аюрведы на русском языке. В основе — классический канон «Аштанга-хридая-самхита» Шримад Вагбхаты (VII в.) в переводе Ю.В. Сорокиной с комментариями И.И. Ветрова. Вокруг текста выстроена полноценная энциклопедическая система: глоссарии, справочники болезней и домашних средств, энциклопедия по 36 разделам, тест конституции и расписание питания.

## О проекте

«Аштанга-хридая-самхита» — один из трёх великих канонов аюрведической медицины. Текст охватывает физиологию, диагностику, терапию, фармакологию, хирургию и духовные практики.

Приложение создано вручную — все материалы извлечены из оригинальных книг, структурированы и адаптированы для чтения на русском языке. Никаких внешних API и баз данных: всё работает как статический сайт прямо из браузера.

## Разделы приложения

### Книга — «Аштанга-хридая-самхита»

Интерактивный читатель основного текста.

- Навигация по **стханам** (разделам) и **главам** в боковой панели, с раскрывающимися группами
- Визуальное разделение **шлок** (стихов), **основного текста** и **комментариев** — каждый тип оформлен отдельно
- **Полнотекстовый поиск** по всем главам в реальном времени с подсветкой совпадений
- **Санскритские термины** подчёркнуты пунктиром — при наведении всплывает тултип с определением и транслитерацией IAST

| Раздел (стхана) | Содержание |
|---|---|
| Сутрастхана | Основные принципы аюрведы |
| Шарирастхана | Анатомия и физиология |
| Нидана стхана | Диагностика болезней |
| Чикитса стхана | Методы лечения |
| Калпасиддхистхана | Рецептура и приготовление |
| Уттара стхана | Специальные разделы |
| Фрагменты Чарака-самхиты | Дополнительные главы |

### Глоссарий санскритских терминов

**382 термина** с определениями, транслитерацией IAST и примерами употребления в тексте.

- Фильтрация по вводу — мгновенный поиск по всему глоссарию
- Термины кликабельны прямо в тексте книги — тултип с подробным определением
- Охватывает: концепции (доша, агни, прана), анатомию, растения, препараты, практики, диагностические методы

- Санскритское название и IAST-транслитерация

### Домашние средства аюрведы

**111 заболеваний** — полная база из книги Васанта Лада «Домашние средства аюрведы».

- Поиск по названию болезни в реальном времени
- Детальная карточка каждого заболевания: описание, составы, рецепты, предупреждения
- Интеллектуальная обработка текста:
  - Составы средств оформлены в маркированные списки
  - Заголовки и подзаголовки внутри статьи выделены типографически
  - Блоки «ВНИМАНИЕ!» и «ПРИМЕЧАНИЕ:» выделены цветным отступом
  - Ссылки вида «(см. Приложение X)» стилизованы как активные
  - Устранены артефакты переноса строк PDF-источника

### Расписание питания

Таблица оптимального времени приёма **106 продуктов** с аюрведической точки зрения.

- Время переваривания в желудке и всасывания в тонком кишечнике
- Поиск по названию продукта
- Фильтрация по категориям (злаки, молочное, фрукты, овощи, специи и др.)
- Показатели: рекомендуемое время суток, доша-эффект, особые условия приёма

### Тест конституции (Пракрити-тест)

Классический аюрведический тест Дипака Чопры / Васанта Лада.

- **60 вопросов** по трём дошам (20 на каждую): Вата, Питта, Капха
- Шкала ответов: 0 (совсем не про меня) — 6 (практически всегда)
- Прогресс-бар в процессе прохождения
- **Результат:**
  - Визуальная диаграмма баланса дош
  - Определение пракрити: моно-, двойная или равновесная конституция
  - Карточка с описанием типа, качеств в балансе и при дисбалансе
  - **6 персональных рекомендаций** по питанию, режиму дня и практикам для каждой доши

## Технический стек

Чистый **HTML / CSS / JavaScript** — никаких фреймворков, сборщиков и зависимостей.

- **ES-модули** — код разбит на независимые `.js` файлы, импортируемые нативно
- **CSS Custom Properties** — полная дизайн-система на переменных; тёмная и светлая тема переключаются одним классом на `<body>`
- **SPA без роутера** — навигация через `history.replaceState` и хэш-фрагменты (`#ch0`, `#glossary`, `#quiz`, `#donate` и т.д.)
- **localStorage** — сохранение темы и позиции чтения
- **Cache-busting** — версионирование модулей через `?v=N` в импортах
- **Tooltip-система** — позиционирование с учётом краёв экрана, задержки наведения и touch-поддержки
- **Парсер текста** — state-machine для обработки PDF/OCR артефактов в «Домашних средствах»: склейка перенесённых строк, детекция ингредиентных списков, предупреждений, подзаголовков

## Запуск локально

Приложение использует ES-модули, поэтому требует HTTP-сервер (не `file://`).

```bash
# Python (встроен в macOS и Linux)
python3 -m http.server 3333

# Node.js
npx serve .

# Затем открыть
open http://localhost:3333
```

## Возможности интерфейса

- **Тёмная / светлая тема** — переключение кнопкой, сохраняется между сессиями
- **Поиск** — по всему тексту книги, работает по всем главам одновременно
- **Прямые ссылки** — каждый раздел доступен по URL с хэшем
- **Адаптация под мобильные** — адаптивная боковая панель
- **Копирование в буфер** — реквизиты на странице доната копируются одним кликом
- **Медицинский дисклеймер** — фиксированный блок в подвале на каждой странице
