## Как добавлять контент безопасно

```bash
# 1. Отредактируй нужный файл
# 2. ОБЯЗАТЕЛЬНО проверь:
node scripts/validate.mjs

# 3. Если всё ОК — коммить:
git add encyclopedia.js
git commit -m "Encyclopedia: add 10 herb articles"
git push
# → Vercel автоматически задеплоит
# → GitHub Actions проверит валидацию
# → (опционально) новый пост выйдет в канал
```
